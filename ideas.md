## Idea 1: Editorial Brutalist

**Design Movement**: Brutalismo Web con influencias de diseño editorial de revistas independientes y fanzines.

**Core Principles**:
1. Contraste extremo entre blanco y negro como lenguaje visual principal
2. Tipografía como elemento gráfico dominante, no solo informativo
3. Asimetría deliberada que rompe la monotonía de las cuadrículas convencionales
4. Espacios vacíos masivos que generan tensión visual y dirigen la mirada

**Color Philosophy**: La paleta se reduce a su esencia: negro (#000000) como autoridad y presencia, blanco (#FFFFFF) como respiración y espacio, y un azul gris oscuro (#33415C) como acento sutil que aparece en momentos clave para guiar la atención sin romper la sobriedad.

**Layout Paradigm**: Composición vertical tipo scroll infinito con secciones que alternan entre pantalla completa (full-bleed) y contenido centrado. Las imágenes se extienden de borde a borde, mientras que los textos se comprimen en columnas estrechas, creando un ritmo visual de expansión y contracción.

**Signature Elements**:
1. Líneas horizontales finas como separadores y elementos de carga/progreso
2. Tipografía en mayúsculas con tracking amplio como identidad visual recurrente
3. Transiciones entre secciones negras y blancas que crean un efecto de "respiración" visual

**Interaction Philosophy**: Las interacciones son sutiles y deliberadas. Los hovers revelan información adicional con transiciones suaves. El scroll es el protagonista, cada sección se revela como una página de revista.

**Animation**: Fade-in suave de elementos al entrar en viewport. Las imágenes aparecen con un ligero efecto de revelación (clip-path). Los textos se deslizan desde abajo con un timing escalonado. Sin animaciones excesivas que distraigan del contenido.

**Typography System**: 
- Display/Títulos: Fuente condensada bold en mayúsculas (ej. "Oswald" o "Bebas Neue") para impacto máximo
- Body: Fuente geométrica limpia (ej. "Space Grotesk" o "DM Sans") para legibilidad
- Detalles: Monoespaciada para elementos técnicos y labels

## Idea 2: Swiss Punk Hybrid

**Design Movement**: Fusión entre el diseño suizo (International Typographic Style) y la estética punk DIY, creando una tensión entre orden y caos controlado.

**Core Principles**:
1. Grid suizo riguroso como base invisible, pero con elementos que lo rompen intencionalmente
2. Fotografía y gráficos hand-drawn coexisten como lenguajes complementarios
3. El texto funciona como textura visual además de contenido legible
4. Cada sección es un "poster" independiente dentro de un flujo continuo

**Color Philosophy**: Monocromático radical. El negro es el lienzo, el blanco es la tinta. No hay grises intermedios excepto los que surgen naturalmente de las fotografías. El color #33415C aparece exclusivamente en elementos interactivos como un guiño sutil de que algo es clickeable.

**Layout Paradigm**: Estructura de "periódico vertical" donde cada sección ocupa el viewport completo. Las imágenes se presentan en formatos variados: full-bleed, centradas con márgenes enormes, o en composiciones de dos columnas asimétricas. El contenido fluye de manera no predecible.

**Signature Elements**:
1. Monograma "CB" como marca de agua recurrente entre secciones
2. Textos descriptivos en tipografía itálica que funcionan como "pies de foto" editoriales
3. Bordes y líneas que aparecen y desaparecen como elementos de composición

**Interaction Philosophy**: Minimalista. El cursor cambia sutilmente sobre elementos interactivos. Los enlaces de navegación (LinkedIn, Instagram, Mail) son funcionales sin adornos. La experiencia es contemplativa, como hojear un libro de arte.

**Animation**: Parallax sutil en las imágenes de fondo. Los textos aparecen con un efecto de máquina de escribir. Las transiciones entre secciones son crossfades lentos. Scroll-triggered animations que revelan contenido progresivamente.

**Typography System**:
- Display: "Clash Display" o "Syne" en pesos extremos (700-900) para títulos impactantes
- Body: "IBM Plex Sans" para texto corrido, limpio y técnico
- Accent: "Space Mono" para labels, fechas y metadatos

## Idea 3: Neo-Constructivist Portfolio

**Design Movement**: Neo-constructivismo digital inspirado en el constructivismo ruso y el diseño gráfico contemporáneo de estudios como Experimental Jetset.

**Core Principles**:
1. Geometría pura como lenguaje: líneas, rectángulos y círculos como elementos estructurales
2. Jerarquía tipográfica extrema con saltos de escala dramáticos
3. El espacio negativo es tan importante como el contenido
4. Cada proyecto se presenta como una composición gráfica autónoma

**Color Philosophy**: Dualidad absoluta. Negro como masa y peso visual, blanco como luz y apertura. El azul gris (#33415C) se reserva para un único elemento por sección, creando un punto focal magnético. La restricción cromática eleva la importancia de cada decisión de color.

**Layout Paradigm**: Columna única que se expande y contrae. Las secciones alternan entre composiciones de texto a la izquierda con imagen a la derecha, y viceversa, creando un zigzag visual. Algunos elementos rompen la columna y se extienden al borde de la pantalla.

**Signature Elements**:
1. Línea de progreso vertical que acompaña el scroll como hilo conductor
2. Números de sección en tipografía gigante semi-transparente como elementos decorativos
3. Transiciones de color (negro a blanco y viceversa) que dividen temáticamente el contenido

**Interaction Philosophy**: Funcional y directa. Los elementos interactivos tienen estados hover claros con inversión de color (blanco sobre negro se convierte en negro sobre blanco). La navegación es intuitiva sin necesidad de menú hamburguesa.

**Animation**: Entrada de elementos con translate-y y opacity. Las secciones se revelan con un efecto de "cortina" vertical. Los textos grandes se animan letra por letra en momentos clave. Smooth scroll entre secciones.

**Typography System**:
- Display: "Archivo Black" o "Anton" para títulos monumentales
- Body: "Work Sans" en peso regular para texto descriptivo
- Technical: "JetBrains Mono" para elementos de interfaz y datos
